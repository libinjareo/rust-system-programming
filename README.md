# rust-system-programming
> 🦀 专注于 **Rust 系统编程** 的系统化学习仓库，包含标准库、内存管理、并发、网络、系统进程、设备 I/O、unsafe 与 FFI 等核心实战内容

## 项目介绍
本仓库由 **libinjareo** 创建，用于从零到一深入学习 **Rust 系统级开发**，不依赖过重框架，聚焦底层、安全、高性能的系统编程能力。

所有代码均可直接编译运行，适合：
- Rust 系统编程入门与进阶
- 学习底层原理、内存安全、系统调用
- 作为学习笔记、面试复习、实战模板

---

## 🎯 学习方向（全覆盖）
- ✅ Rust 标准库深度使用
- ✅ 环境管理、路径与配置处理
- ✅ 命令行工具（CLI）开发
- ✅ 内存管理、所有权、生命周期、智能指针
- ✅ 网络编程（TCP/UDP/Socket）
- ✅ 并发与多线程安全
- ✅ 系统进程创建、管理与信号处理
- ✅ 设备 I/O、文件系统、底层 IO
- ✅ 不安全 Rust（unsafe）编程
- ✅ FFI 与 C/C++ 交互

## 🚀 快速开始
### 1. 安装 Rust
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### 2. 克隆项目
```bash
git clone https://github.com/libinjareo/rust-system-programming.git
cd rust-system-programming
```

### 3. 运行示例
```bash
# 运行所有示例
cargo run

# 运行指定模块示例
cargo run --bin 文件名
```

### 4. 运行测试
```bash
cargo test
```
